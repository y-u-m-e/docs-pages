# Yume Tools Documentation

Welcome to the Yume Tools documentation! This is your comprehensive guide to the OSRS clan management platform.

## Overview

Yume Tools is a comprehensive OSRS clan management platform consisting of:

- **Unified API Gateway** (`api.emuy.gg`) - Routes to microservices
- **Central Authentication** - Discord OAuth with RBAC
- **Attendance Tracking** - Cruddy Panel for event attendance
- **Tile Events** - Snake-style progression events (Ironforged Events)
- **Bingo Tracker** - Competition tracking with RuneLite plugin
- **OSRS Data Integration** - WOM, GroupIron.men, RuneProfile

## Features

### For Clan Members
- **Event Participation** - Join tile events and bingo competitions
- **Progress Tracking** - View your achievements and leaderboard standings
- **Attendance Records** - Track your event participation history

### For Clan Leaders
- **Event Management** - Create and manage tile events and bingo competitions
- **Attendance Tracking** - Record and export attendance data
- **Role Management** - Assign permissions with granular RBAC

### For Developers
- **RESTful APIs** - Well-documented endpoints for all services
- **RuneLite Plugin** - Automatic achievement tracking
- **Webhooks** - Real-time event notifications
