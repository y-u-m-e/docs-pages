# Yume Tools Documentation

Welcome to the Yume Tools documentation!

## Overview

Yume Tools is a comprehensive OSRS clan management platform consisting of:

- **Unified API Gateway** (`api.emuy.gg`) - Routes to microservices
- **Central Authentication** - Discord OAuth with RBAC
- **Attendance Tracking** - Cruddy Panel for event attendance
- **Tile Events** - Snake-style progression events (Ironforged Events)
- **Bingo Tracker** - Competition tracking with RuneLite plugin
- **OSRS Data Integration** - WOM, GroupIron.men, RuneProfile
